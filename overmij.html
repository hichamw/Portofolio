Persoonlijke kwaliteiten en kenmerken:

Ik zit nu in het eerste leerjaar van mijn studie Informatica aan de Hogeschool Rotterdam. Ik heb tot nu toe gewerkt met:

  • Javascript
  • Java
  • Greenfoot

Ik heb tot nu toe heel veel geleerd van niet alleen programmeren, maar ook dingen zoals website analyse, testen van systemen en UML. Ik heb het heel erg naar m'n zin op deze opleiding, en ik ben heel benieuwd naar wat de toekomst gaat brengen.


Interesse gebieden:

Ik ben niet heel zeker over waar mijn interesses precies liggen, omdat ik nog niet met de C talen heb gewerkt of applicatiedevelopment. Maar ik kan me wel een beetje vinden in Java, object georiënteerd programmeren vind ik wel redelijk interessant. Op zich is webdevelopment ook wel leuk, maar er zit minder uitdaging in.


Toekomstplannen:

Zoals ik in de vorige paragraaf al heb gezegd, ik heb nog niet helemaal een idee van wat ik wil gaan doen later, maar als ik mocht kiezen dan zou ik waarschijnlijk voor gamedevelopment gaan. Ik zou dat kiezen omdat ik geinteresseerd ben in game-mechanismen en hoe games in elkaar zitten. Dus die richting zou waarschijnlijk wel goed bij mij passen.
